Deep Recurrent Attentive Writer for 1 dimensional data

This post takes our implementation for DRAW and rewrites them for 1 dimensional signals.

Plots

The UCR database supplies with applicable time series to show our implementation on. For this experiment, we chose the ECG5000 dataset. Note that the ECG's run from R-peak to R-peak.

  • The left column displays the original signal and the read bar
  • The __middle_ column displays the current Gaussians about to be written
  • The right column displays the written signal plus the write bar

Every time-step, the network write to the canvas in the right columns. In this writing operation, we add a mixture of Gaussians to the canvas. Gaussians need only two parameters, which makes them suited for this task. The middle columns plots in blue the five individual Gaussians. The red curve is the summation of the five Gaussians.

In the original post we discuss some context of the DRAW and refer to additional material.
